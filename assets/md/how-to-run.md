# How to build and run

## Build

To build the project, run the following command:

```bash
    make build
```

or

```bash
    mvn clean package
```

## Run

To run it simply use commands from below

```bash
    make run
```

or

```bash
    java -jar target/fractal-flame-1.0.0.jar
```
